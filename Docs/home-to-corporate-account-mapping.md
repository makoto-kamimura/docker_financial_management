# 家庭モード → 法人モード 勘定科目対応表

家庭モード（`H-` 系）の勘定科目を法人会計に転用する際の対応関係と、  
必要な調整内容をまとめます。

---

## 凡例（調整区分）

| 記号 | 意味 |
| ---- | ---- |
| ✅  | ほぼそのまま使用可（科目名のみ読み替え） |
| 🔄  | 科目の統合・細分化が必要 |
| ↩️  | 収益↔費用の向きが逆になる |
| ⚠️  | 税務上の取り扱いに注意が必要 |
| ❌  | 法人経費として原則認定不可 |

---

## 調整区分サマリー

| 調整区分     | 対象科目数 | 主な対象                                         |
| ------------ | ---------- | ------------------------------------------------ |
| ✅ そのまま  | 7          | 株式配当・保険割戻金・売上系・消耗品費等         |
| 🔄 統合/細分化 | 12         | 光熱費3科目→水道光熱費、通信費2科目→通信費 等   |
| ↩️ 向き逆転  | 1          | 給与（家庭:収入 → 法人:費用）                   |
| ⚠️ 税務注意  | 11         | 接待交際費・保険料・減価償却対象資産 等          |
| ❌ 経費不可  | 4          | ペット関連3科目・理容/美容費                     |

## 実装時の注意点

1. **モード切替時のマッピング**: 家庭モードから法人モードへ切り替える際は、上記対応表をもとに科目コードの変換ロジックが必要。
2. **按分計算**: 自動車保険・モバイル回線等、私用兼業の科目は事業使用割合を別途管理する仕組みが必要。
3. **固定資産しきい値**: `H-3016 家具家電設備費` は金額（10万円）を基準に科目と処理が分岐するため、仕訳入力時に金額チェックロジックを設ける。
4. **経費不可科目の扱い**: `❌` 科目は法人モードでは非表示にするか、警告を表示する UI 制御を検討。
