# 勘定科目変換マスタ

家庭モード（`H-` 系）の全勘定科目について、**個人事業主モード**および**法人モード**への  
変換先科目・注意事項・調整区分を一覧化したマスタです。

---

## 凡例（調整区分）

| 記号 | 意味 |
| ---- | ---- |
| ✅  | 科目名の読み替えのみ（ほぼそのまま使用可） |
| 🔄  | 統合・細分化が必要（複数→1科目 or 1科目→複数） |
| ↩️  | 収益↔費用の向きが逆になる |
| ⚠️  | 税務上の注意が必要（按分・損金制限・控除区分等） |
| ❌  | 経費として原則認定不可 / 帳簿外で管理 |

## 6. 個人事業主特有の注意事項

1. **家事按分**: 自宅兼事務所の光熱費・家賃・通信費は事業使用割合（面積比・時間比・接続数比等）で按分する必要があります。按分割合は根拠資料（間取り図・利用記録等）を保存してください。
2. **事業主貸・事業主借**: 個人事業主が事業資金から生活費を支出した場合は「事業主貸」、個人資金を事業に充当した場合は「事業主借」を使用します。給与の概念は存在しません。
3. **元入金**: 個人事業主の自己資本は「元入金」で管理します。法人の資本金に相当します。
4. **青色申告特別控除**: 青色申告では最大65万円の所得控除が受けられます。帳簿の正確な記帳が前提です。
5. **固定資産の按分**: 車・PCなど事業と私用を兼用する資産は、事業使用割合で減価償却費を按分します。

## 7. 法人特有の注意事項

1. **役員報酬の損金算入制限**: 役員報酬は原則として定期同額給与・事前確定届出給与のみ損金算入可。業績連動給与は一定要件あり。
2. **接待交際費の損金算入制限**: 中小法人（資本金1億円以下）は年間800万円以下 or 接待飲食費の50%のいずれか選択。
3. **受取配当等の益金不算入**: 完全子会社からの配当は全額、関連法人株式等は一部益金不算入。
4. **少額減価償却資産の特例**: 中小企業は取得価額30万円未満の資産を即時費用計上可（年300万円上限）。
5. **消費税区分**: 各科目に消費税の課税区分（課税/非課税/免税/不課税）を設定する必要があります。
