# 家庭モード 勘定科目一覧

家庭（個人）向けモードで使用する勘定科目の定義です。  
コードは `H` プレフィックスで始まり、ビジネスモードの勘定科目（`1000`〜`8300` 番台）と区別します。

---

## 経過勘定（ASSET / LIABILITY）

発生主義を適用する際に使用する経過勘定科目です。  
詳細は [home-mode-accrual-basis.md](./home-mode-accrual-basis.md) を参照してください。

| コード | 勘定科目名 | カテゴリ | 説明 |
| ------ | ---------- | -------- | ---- |
| H-5001 | 前払費用   | ASSET    | 支払済みだがまだ費用化していない金額（保険料・年会費等の先払い分） |
| H-5002 | 未払費用   | LIABILITY | 費用は発生したがまだ支払っていない金額（光熱費・クレジット請求待ち等） |
| H-5003 | 未収収益   | ASSET    | 収益は発生したがまだ受け取っていない金額（給与・配当等の未入金分） |
| H-5004 | 前受収益   | LIABILITY | 受取済みだがまだ収益化していない金額（前受家賃・前払サービス料等） |

## カテゴリ対応表

| AccountCategory | 対象コード範囲         | 用途                                         |
| --------------- | ---------------------- | -------------------------------------------- |
| `REVENUE`       | H-1001 〜 H-1016       | 収入（給与・配当・給付金等）                 |
| `COGS`          | H-2001 〜 H-2017       | 変動生活費（食費・日用品等）                 |
| `EXPENSE`       | H-3001 〜 H-3034       | 固定費・経費（光熱費・保険等）               |
| `ASSET`         | H-5001, H-5003         | 経過勘定・資産（前払費用・未収収益）         |
| `LIABILITY`     | H-4001 〜 H-4005       | 負債・ローン残高                             |
| `LIABILITY`     | H-5002, H-5004         | 経過勘定・負債（未払費用・前受収益）         |

---

## 備考

- `ASSET`（資産）は銀行口座・投資口座・現金等を `LinkedAccount` または `BankAccount` で管理するため本一覧には含みません。
- ローン（H-4001〜H-4005）はローン残高を `LIABILITY` として管理し、月次の返済額は `Loan` / `LoanRepayment` モデルで追跡します。
- 個人事業主・法人モードへの変換対応は [account-master-mapping.md](./account-master-mapping.md) を参照してください。
