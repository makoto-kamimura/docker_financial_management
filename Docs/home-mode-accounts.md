# 家庭モード 勘定科目一覧

家庭（個人）向けモードで使用する勘定科目の定義です。  
コードは `H` プレフィックスで始まり、ビジネスモードの勘定科目（`1000`〜`8300` 番台）と区別します。

---

## 収入（REVENUE）

| コード  | 勘定科目名   | 説明                                   |
| ------- | ------------ | -------------------------------------- |
| H-1001  | 給与         | 勤務先から受け取る給与・賞与           |
| H-1002  | 株式配当     | 保有株式から得られる配当金             |
| H-1003  | 保険割戻金   | 保険会社からの割戻金・キャッシュバック |
| H-1004  | 事業所得     | 個人事業・副業・フリーランス収入       |
| H-1005  | 不動産所得   | 賃貸物件・駐車場等の家賃・地代収入     |

---

## 変動生活費（COGS）

日々の生活で変動する支出を管理します。

| コード  | 勘定科目名       | 説明                                     |
| ------- | ---------------- | ---------------------------------------- |
| H-2001  | 食費                   | 食料品（主食・副食・調味料等）の購入費用       |
| H-2002  | 飲料費                 | 飲料品（水・ジュース・酒類等）の購入費用       |
| H-2003  | 健康食費               | サプリメント・健康食品・プロテイン等           |
| H-2004  | 日用品/消耗品          | 洗剤・トイレットペーパー・雑貨等の消耗品       |
| H-2005  | 医療費                 | 病院・薬局・歯科等の医療費用                   |
| H-2006  | ペットの飲食費         | ペットフード・おやつ等の飲食費用               |
| H-2007  | ペットの医療費         | 動物病院・ワクチン・薬等の医療費用             |
| H-2008  | ペットの日用品/消耗品  | ペット用トイレ用品・ケア用品・おもちゃ等       |

---

## 固定費・経費（EXPENSE）

毎月ほぼ一定額が発生する支出を管理します。

### 公共料金・通信費

| コード  | 勘定科目名         | 説明                               |
| ------- | ------------------ | ---------------------------------- |
| H-3001  | 電気代             | 電力会社への支払い                 |
| H-3002  | ガス代             | ガス会社への支払い                 |
| H-3003  | 水道代             | 水道局への支払い                   |
| H-3004  | 固定ネット回線     | 光回線・CATV 等の固定インターネット |
| H-3005  | モバイルネット回線 | スマートフォン・モバイル通信費     |

### 交通費

| コード  | 勘定科目名     | 説明                                   |
| ------- | -------------- | -------------------------------------- |
| H-3006  | 通勤費         | 定期代・通勤に要する交通費             |
| H-3007  | その他交通費   | 通勤以外の電車・バス・タクシー代等     |

### 娯楽・外食費

| コード  | 勘定科目名     | 説明                                   |
| ------- | -------------- | -------------------------------------- |
| H-3008  | 娯楽/外食費    | 外食・映画・旅行・趣味等の娯楽支出     |

### 保険料

| コード  | 勘定科目名   | 説明                                         |
| ------- | ------------ | -------------------------------------------- |
| H-3009  | 社会保険     | 健康保険料・厚生年金・雇用保険等の社会保険料 |
| H-3010  | 医療保険     | 民間の医療保険・がん保険等の保険料           |
| H-3011  | 自動車保険   | 任意自動車保険・車両保険の保険料             |

### 税金

| コード  | 勘定科目名   | 説明                                             |
| ------- | ------------ | ------------------------------------------------ |
| H-3012  | 自動車税     | 自動車税・軽自動車税の年額                       |
| H-3013  | 固定資産税   | 土地・建物・償却資産に課される固定資産税         |
| H-3014  | 住民税       | 都道府県民税・市区町村民税（特別徴収・普通徴収） |

### その他固定費

| コード  | 勘定科目名       | 説明                                           |
| ------- | ---------------- | ---------------------------------------------- |
| H-3015  | 事業経費         | 個人事業・副業に要する経費（仕入・消耗品等）   |
| H-3016  | 家具家電設備費   | 家具・家電・設備の購入・修繕費用               |
| H-3017  | 理容/美容費      | 散髪・美容院・エステ・化粧品等の費用           |

## 負債・ローン（LIABILITY）

| コード  | 勘定科目名     | 説明                                       |
| ------- | -------------- | ------------------------------------------ |
| H-4001  | 住宅ローン     | 住宅購入・リフォームのための借入残高       |
| H-4002  | 自動車ローン   | 自動車購入のための借入残高                 |

---

## カテゴリ対応表

| AccountCategory | 対象コード範囲 | 用途                   |
| --------------- | -------------- | ---------------------- |
| `REVENUE`       | H-1xxx         | 収入                   |
| `COGS`          | H-2xxx         | 変動生活費             |
| `EXPENSE`       | H-3xxx         | 固定費・経費           |
| `LIABILITY`     | H-4xxx         | 負債・ローン残高       |

---

## 備考

- `ASSET`（資産）は銀行口座・投資口座・現金等を既存の `LinkedAccount` または `BankAccount` で管理するため、本一覧には含みません。
- ローン（`H-4001`・`H-4002`）はローン残高を `LIABILITY` として管理し、月次の返済額は `EXPENSE` ではなく `Loan` / `LoanRepayment` モデルで追跡します。
- seed ファイルとして実装する場合は `app/web/prisma/seed-home-accounts.ts` を参照してください（未作成の場合は本ドキュメントをもとに作成してください）。
