# 個人事業主向け決算・確定申告支援システム 仕様書

## 1. システム概要

### システム名

個人事業主向け決算・確定申告支援システム

### 目的

個人事業主が年間の会計データを管理し、決算処理および確定申告に必要な情報を効率的に作成・管理できるシステムを提供する。

### 対象ユーザー

* 個人事業主
* フリーランス
* 小規模事業者
* 移動販売事業者（コーヒースタンド等）

# 5. 機能詳細

## F001 ユーザー管理

### 概要

事業者情報管理

### 項目

* 屋号
* 氏名
* 開業日
* 青色申告区分
* インボイス登録番号
* 消費税課税方式

---

## F002 取引管理

### 登録項目

| 項目   | 内容        |
| ---- | --------- |
| 取引日  | 日付        |
| 摘要   | 内容        |
| 金額   | 数値        |
| 勘定科目 | 選択        |
| 税区分  | 課税/非課税    |
| 支払方法 | 現金・銀行・カード |

## F006 棚卸管理

### 登録項目

* 商品名
* 数量
* 単価
* 在庫金額

### 計算

```text
期末在庫金額
= 数量 × 単価
```

### 対象例

コーヒースタンド

* コーヒー豆
* 紙カップ
* シロップ
* お菓子

---

## F007 固定資産管理

### 項目

* 資産名
* 取得日
* 取得金額
* 耐用年数
* 償却方法

### 対象

* エスプレッソマシン
* グラインダー
* 冷蔵庫
* パソコン
* 車両

---

## F008 減価償却計算

### 対応

* 定額法
* 定率法

### 自動計算

```text
年間償却費
= 取得価額 ÷ 耐用年数
```

---

## F009 家事按分管理

### 管理対象

* 家賃
* 電気代
* 通信費
* 車両費

### 設定

```text
通信費
事業利用率 70%
```

### 自動計算

```text
経費計上額
= 支払額 × 利用率
```
