# 個人事業主・法人対応 会計・決算・申告支援システム 要件定義書

## 1. システム概要

### システム名称

統合会計・決算・税務申告支援システム

### 目的

個人事業主および法人が、日常会計から決算・税務申告までを一元管理できるクラウドシステムを提供する。

### 対象事業者

* 個人事業主
* フリーランス
* 株式会社
* 合同会社
* 小規模法人
* 店舗事業
* 移動販売事業
* EC事業
