FAMCO

おうちCEOになろう。

FAMCOは、「経営者視点を家計簿から学ぶ」をコンセプトにした、新しい家庭経営レポートアプリです。

家計簿は、ただ支出を記録するだけではありません。

会社が毎年決算を行い、利益や資産の状況を確認するように、家庭にも「経営」という視点があります。

FAMCOでは、収入と支出を記録するだけでなく、

* 毎月の利益（貯蓄額）
* 家庭の資産と負債
* 純資産の推移
* 将来に向けた資産形成

を、経営レポートのようにわかりやすく可視化します。

「今月はいくら使ったか」ではなく、

「わが家の価値はどれだけ成長したか」

を確認できるのがFAMCOの特徴です。

家計簿から経営感覚を身につける

家庭は小さな会社です。

収入は売上、
生活費は経費、
貯蓄は利益、
資産は会社の財産。

FAMCOは、家計を経営の視点で捉えることで、お金の使い方や資産形成について自然と学べる仕組みを提供します。

主な機能

* 収入・支出の記録
* 家庭版損益計算書（P/L）
* 家庭版貸借対照表（B/S）
* 純資産レポート
* 資産推移グラフ
* 月次・年次決算レポート
* 家族で共有できる経営ダッシュボード

こんな方におすすめ

* 家計簿が続かない方
* 貯蓄や資産形成を始めたい方
* 投資やお金の勉強をしたい方
* 経営感覚を身につけたい方
* 家族のお金を見える化したい方

FAMCOは、家計管理アプリではありません。

あなたと家族が「おうちCEO」として、未来の暮らしを経営するための家庭経営レポートアプリです。
